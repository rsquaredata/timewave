## Model comparison framework

| **Model category**          | **Model**            | **Tuning / Hyperparameters**                                   | **Scope (search space / strategy)**                                                              |
|-----------------------------|----------------------|----------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| **Baselines**               | SNaive               | –                                                              | Reference model only (no tuning).                                                                |
|                             | RW w/ Drift          | –                                                              | Reference model only (no tuning).                                                                |
| **Deterministic**           | SES                  | $\alpha$ (level)                                               | ETS(A,N,N);<br>$\alpha \in \\{0.001,0.5\\}$.                           |
|                             | Holt Linear Trend    | $\alpha$ (level), $\beta$ (trend), $\phi$ (damping)            | $\alpha,\beta\in\\{0.001,0.20\\}$;<br>$\phi \in \\{\text{TRUE},\text{FALSE}\\}$;<br>selected by rolling CV            |
|                             | Holt-Winters          | $\alpha$ (level), $\beta$ (trend), $\gamma$ (seasonality)      | $\alpha,\beta\in\\{0.001,0.06\\}$;<br>$\gamma\in\\{0.015,0.25\\}$;<br>`seasonal`$\in\\{\text{additive},\text{multiplicative}\\}$;<br>residual diagnostics (ACF, Ljung–Box).<br>Rolling CV             |
| **Stochastic**              | `auto.arima()`       | `stepwise`, `approximation`, `seasonal`                        | `stepwise` $\in\\{\text{TRUE},\text{FALSE}\\}$, `approximation` $\in\\{\text{TRUE},\text{FALSE}\\}$;<br>shortlist by CV. |
|                             | ARIMA                | $(p,d,q)$ (non seasonal orders)                                | $p,q \in [0..5]$, $d \in \\{0,1\\}$;<br>AICc pre-filtering then rolling CV.                        |
|                             | SARIMA               | $(p,d,q)$, $(P,D,Q)_s$                                         | $P,Q \in [0..2]$, $D \in \\{0,1\\}$, $s=96$;<br>bounded total order.                               |
|                             | SARIMAX              | $(p,d,q)$, $(P,D,Q)_s$, `xreg`                                 | `xreg`: $\text{Temp}_t$, Temp lags ($t-1 \ldots t-96$), optional rolling Temp;<br>CV-based selection. |
| **Regression-based models** | Quadratic Regression | feature choices                                                | Trend | $t^2$;<br>daily cycle via $\sin/\cos$ ($k=1..5$);<br>Temp (| lags).                       |
|                             | NNAR                 | `p`, `P`, `size`, `repeats`                                    | `p` $\in \\{1..96\\}$, `P` $\in \\{0,1,2\\}$;<br>`size` $\in \\{5,10,20,40\\}$;<br>`repeats` $\in \\{20,50\\}$ |
| **Machine Learning**        | Random Forest        | `mtry`, `ntree`, `nodesize`                                    | `mtry` $\in \\{\sqrt{p},\frac{p}{3},\frac{p}{3}\\}$;<br>`ntree` $\in \\{500,1000,2000\\}$;<br>`nodesize` $\in \\{1,5,10\\}$. |
|                             | XGBoost              | `eta`, `max_depth`, `nrounds`, `subsample`                     | `eta` $\in \\{0.01,0.05,0.1\\}$;<br>`max_depth` $\in \\{3,5,7,10\\}$;<br>`subsample` $\in \\{0.6,0.8,1\\}$;<br>early stopping. |
|                             | SVM / SVR            | `kernel`, `cost`, `gamma`, `epsilon`                           | `kernel = "rbf"`;<br>`cost` $\in 2^{[-5..15]}$;<br>`gamma` $\in 2^{[-15..3]}$;<br>`epsilon` $\in \\{0.01,0.05,0.1\\}$. |
|                             | Prophet              | `changepoint_prior_scale`, `seasonality_mode`, `fourier_order` | `changepoint_prior_scale` $\in \\{0.01,0.05,0.1,0.5\\}$;<br>`seasonality_mode` $\in \\{\text{"additive"},\text{"multiplicative"}\\}$;<br>`fourier_order` $\in \\{5,10,15,20\\}$;<br>Temp as regressor. |
| **Distance-based**          | WNN                  | $k$, distance, weighting                                       | $k \in \\{1..50\\}$ + adaptive $\sqrt{N}$;<br>distance $\in\\{\text{euclidean},\text{manhattan}\\}$;<br> weights $\in\\{\text{uniform},\frac{1}{d}\\}$.  |
